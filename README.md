# taby

It's a taboo game.

## Tech and Libraries

- MVVM Architecture
- State Management: [Provider](https://pub.dev/packages/provider)
- Database: [Hive](https://pub.dev/packages/hive)
- Ads: [Google Mobile Ads](https://pub.dev/packages/google_mobile_ads)
